# Lunar Dark

Lunar Dark is a custom dark theme for VS Code and code-server with bright neon accents, high-contrast syntax highlighting, and a moonlit teal identity.

## Included Theme

- `Lunar Dark`

## Development

Package the extension with:

```bash
npx @vscode/vsce package
```

Install the resulting `.vsix` with:

```bash
code-server --install-extension shared-assets/vscode-themes/lunar-dark-0.0.1.vsix
```
