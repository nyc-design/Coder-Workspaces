Container workspace running code-server in the browser, connecting to a bare VM over SSH. Optional SSHFS mount maps remote VM files to /workspaces/remote-vm so you can browse and edit them through the browser IDE without installing local VS Code. Use this when you want a thin browser-only IDE for an existing remote VM (no local Mac/Windows install needed). Provide the bare VM's host, user, and SSH port; SSH key auth via Coder's external SSH key. Not for general dev work — use project-workspace for that.
