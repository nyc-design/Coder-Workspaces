Persistent Windows VM on Google Cloud Compute Engine, surfaced as a browser-based desktop in Coder via Apache Guacamole RDP. Use when you need a Windows-native dev/test environment for software that won't run on Linux containers. Workspace start sets the GCE VM to RUNNING; stop sets it to TERMINATED to cut compute billing while keeping the persistent boot disk + data disk intact. Default sizing is e2-standard-2 / 50GB boot disk for occasional use; bump CPU/memory/disk parameters for heavier workloads. Multi-arch Guacamole image works on both ARM and AMD64 Coder hosts. Not for general coding work — use project-workspace for that.
