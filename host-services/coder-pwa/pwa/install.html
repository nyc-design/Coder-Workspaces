Install Coder

Add Coder to your home screen so workspaces open in a full-screen app instead of a browser tab.

Android & Desktop Chrome

Checking install availability…

iPad / iPhone Safari

  1. Tap the Share button at the top of Safari.
  2. Scroll and choose Add to Home Screen.
  3. Tap Add — the Coder icon appears on your home screen.

Other browsers

Open this page in Chrome, Edge, or Safari on the device where you want the app installed. Look for an Install / Add to Home Screen option in the browser menu.

Once installed, launching the app goes straight to coder.tapiavala.com.
